💻 硬體需求

AMD AI MAX+ 395 建議配置

項目 規格 說明
CPU AMD Ryzen AI MAX+ 395 16 Zen 5 核心
GPU 40 CU RDNA 3.5 Radeon 8060S integrated
記憶體 128GB unified ⚠️ 設定 64GB/64GB
作業系統 Windows 10/11 推薦 Windows 11
儲存空間 50GB+ 模型 + 依賴 + 輸出
⚠️

RAM 配置注意事項

必須在 BIOS 中設定為 64GB/64GB (CPU/GPU 均分)

預設的 96GB/32GB 會導致 OOM 錯誤！

🛠️ 安裝步驟

四步驟完成設定

1

安裝 AMD Adrenaline 驅動

  1. 前往 AMD 官方頁面
  2. 下載最新版 Adrenaline 驅動
  3. 安裝後重新啟動電腦
2

安裝 Git

如已安裝可跳過。

  1. 前往 Git 官網
  2. 下載 Windows 版本並安裝
  3. 驗證：git --version
3

安裝 ComfyUI Desktop（核心）

✨

自動安裝 Python、ROCm 7.2.2、PyTorch，無需手動配置！

  1. 前往 ComfyUI 官方 Blog
  2. 下載 Windows 版本
  3. 執行安裝（全部預設選項）
  4. ⏱️ 首次啟動需 10-20 分鐘
4

調整記憶體分配

⚠️

必須設定！不設定會 OOM

  1. 重啟進入 BIOS（通常按 Delete 或 F2）
  2. 找到 Memory Configuration
  3. 設定為 64GB/64GB
  4. 儲存並重啟

🎨 使用教學

節點式工作流程快速上手

快速開始：使用預設範本

1

載入範本：File → Load → Templates → LTX Video

2

找到 "Text Prompt" 節點，輸入描述

一隻可愛的橘貓在陽光下打哈欠
3

設定參數：Duration 5秒、Resolution 1920x1080

4

點擊 Queue Prompt 開始生成

5

等待 3-5 分鐘，完成後在 ComfyUI\output\ 查看

🐌 生成太慢？

  • 降低解析度 → 1280x720
  • 縮短時長 → 3 秒
  • 減少步數 → Steps 20

💥 遇到 OOM？

  • 確認 RAM 64GB/64GB
  • 關閉其他程式
  • 降低解析度或時長

✨ 想要更高品質？

  • 增加步數 → Steps 40-50
  • 調整 CFG Scale → 7-9
  • 寫更詳細的 prompt

❓ 常見問題

快速找到解答

首次啟動很慢，正常嗎？

✅ 正常。首次啟動需載入模型和初始化 ROCm，約 1-3 分鐘。之後啟動會快很多（約 30 秒）。

出現 "CUDA out of memory" 錯誤？
  1. 確認 RAM 配置為 64GB/64GB（不是 96GB/32GB）
  2. 降低解析度或影片長度
  3. 關閉其他占用記憶體的程式
生成速度比預期慢？

AMD AI MAX+ 395 預期約 3-5 分鐘生成 5 秒影片。如果更慢：

  • 檢查是否有其他程式占用 GPU
  • 確認驅動已正確安裝
  • 查看工作管理員 GPU 使用率
可以生成超過 5 秒的影片嗎？

可以，但：

  • 越長越耗時（10 秒約 6-8 分鐘）
  • 越長越容易 OOM（建議不超過 10 秒）
  • 可分段生成再用影片編輯軟體合併
我的 AMD GPU 不是 AI MAX+ 395，能用嗎？

✅ 可以！已驗證支援：

  • AMD 7900 XTX (24GB VRAM)
  • AMD 7800 XT (16GB VRAM)
  • AMD Strix Halo 系列

更多請參考 ROCm 支援清單
