Quick Start Workflow

  1. 開啟 ComfyUI Desktop，載入 LTX 2.3 範本工作流。
  2. 填入 Prompt、長度（建議先 3-5 秒）、解析度（先 720p）。
  3. 按 Queue Prompt 執行，觀察 VRAM/RAM 使用情況。
  4. 輸出影片位於 ComfyUI output 目錄，確認結果後再逐步提高品質。

FAQ

1) 首次啟動很慢，正常嗎？

正常，首次會初始化環境與模型，後續啟動時間會明顯下降。

2) OOM 要先檢查什麼？

先確認 BIOS 為 64GB/64GB，再降低影片長度與解析度測試。

3) 非 AI MAX+ 395 也能用嗎？

可，需確認你的 AMD GPU 在 ROCm 支援範圍內並使用對應驅動。

4) 為何建議先 720p、3-5 秒？

可先驗證流程與穩定性，再逐步提高品質，降低排錯成本。

5) 還有 CLI 路線嗎？

可以，參考 ltx-cli 專案做批次化或腳本化流程。
